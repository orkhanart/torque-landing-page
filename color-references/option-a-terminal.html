Color Philosophy

The Terminal approach treats the interface as a data environment where color is precious and meaningful. The monochrome base (pure blacks, whites, grays) creates a neutral canvas that lets data speak. The single accent color (Citrine) becomes the signal - it means "look here," "act now," or "this matters."

This approach draws from Bloomberg terminals, code editors, and professional trading interfaces where information density is high and distraction is costly.
